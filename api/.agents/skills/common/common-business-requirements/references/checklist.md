# BRD Checklist

- [ ] Business objective is measurable (baseline + target + date).
- [ ] Sponsor and decision owners identified.
- [ ] AS-IS pain and TO-BE outcomes documented.
- [ ] Scope and non-scope explicitly listed.
- [ ] Assumptions and constraints listed.
- [ ] Risk register includes mitigation owner.
- [ ] Value statement is tied to cost, revenue, risk, or compliance.
- [ ] Handoff links to candidate PRD requirements exist.
